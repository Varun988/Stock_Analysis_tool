# Stock Analysis Tool

> Educational investment recommendation and portfolio analysis tool for Indian ETFs, mutual funds, and future investment options.

## Overview

This project is being rebuilt as a clean, portfolio-aware investment recommendation tool.

The tool will help users make disciplined long-term investment decisions by combining:

- User investment profile
- Monthly investment amount
- Risk appetite
- Time horizon
- Uploaded portfolio statement/CAS data
- Historical market data
- Portfolio diversification analysis
- Rule-based recommendation scoring
- Beginner-friendly AI explanations

## Important Disclaimer

This application is for educational and decision-support purposes only. It does not provide guaranteed financial advice. Past performance does not guarantee future returns. All investments are subject to market risk.

## MVP Focus

The MVP will focus on:

- Indian ETFs
- Indian mutual funds
- Monthly investment recommendations
- Statement/CAS-based portfolio import
- Portfolio-aware recommendation logic
- Web app first
- GitHub Codespaces-based development

## Development Environment

This project is designed to run in GitHub Codespaces so that no local Python or Node.js module installation is required on the developer machine.
